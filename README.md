# Razorpay Clone (Frontend)

This project is a frontend clone of the Razorpay website built using HTML and Tailwind CSS. It replicates the design and layout of the original platform for learning and practice purposes.

## 🚀 Features

* Razorpay-inspired UI design
* Structured layout with multiple sections (Navbar, Hero, Features, etc.)
* Built using utility-first CSS (Tailwind CSS)
* Clean and readable code structure

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* PostCSS

```text
## 📁 Project Structure

razorpay-clone/

│
└── node_modules/           # Dependencies (ignored)
```

## 📌 Note

This is a frontend-only project. No backend functionality or payment processing is implemented. All buttons and links are non-functional and serve only as UI placeholders.

## 📚 Learning Outcomes

- Understanding of modern UI development
- Working with Tailwind CSS and utility-first styling
- Basic project structuring for frontend applications
- Responsive design principles
- CSS Grid and Flexbox layouts

## ⚙️ Setup Instructions

1. Clone the repository
2. Install dependencies:
   npm install

3. Run the project:
   npm run start
